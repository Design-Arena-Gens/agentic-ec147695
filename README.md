# कृष्ण-विदुर प्रसंग — Next.js साइट

यह छोटी-सी साइट महाभारत-प्रेरित प्रश्न का उत्तर संक्षेप में प्रस्तुत करती है: 
श्रीकृष्ण ने हस्तिनापुर में विदुर के घर ही भोजन क्यों किया?

## स्थानीय रूप से चलाएँ

```bash
npm install
npm run dev
```

## उत्पादन बिल्ड

```bash
npm run build
npm start
```

## Vercel पर परिनियोजन

परियोजना कॉन्फ़िगरेशन Next.js के लिए मानक है; `vercel` CLI से सीधे परिनियोजित करें।
